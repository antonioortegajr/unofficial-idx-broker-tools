Site Analyzer

This tool does the following:

* Mobile friendliness
* Google page Speed score
* Checks for iframes
* Creates link to check for number of indexed pages


## Run on Heroku 
The source can be run on [Heroku](https://www.heroku.com/) using the free service level.
